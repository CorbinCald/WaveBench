How dark should the foreground layers be?

Darker layers recede more convincingly but lose color detail. Each option darkens progressively; you're picking how steep the ramp is. Shown in green (default theme) — the same ratios apply to every theme.

A

Gentle ramp — 100% / 65% / 40% ← recommended

Each layer multiplies the main wave color by a brightness factor. Still recognizably "the theme color", just dimmer. Matches shadow behavior where the shadow is a tinted version of the lit surface.

B

Steep ramp — 100% / 50% / 25%

More dramatic depth, but layer 2 gets close to background-black. Good for "night ocean" feel; foreground layers become near-silhouettes against the sky above.

C

Theme-calibrated dark steps

Instead of a flat multiplier, pull from the theme's existing title_wave gradient — which is already hand-tuned per theme. Layer 1 uses index 5-6, layer 2 uses index 2-3. Uses the same dark values you already picked for title waves.

Trade-off: A is simpler to implement (one multiplier), works uniformly across all themes, and you can tune the numbers later. C reuses existing hand-tuned color stops, but couples the foreground layers to whatever the title_wave designer picked.
