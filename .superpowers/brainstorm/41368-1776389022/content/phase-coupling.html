How tightly should layers follow the main wave?

You said "slightly independent but shadow the crests and valleys." That's a spectrum — here's how three points on it behave. The key knob is phase coupling: how much each layer's surface is driven by the main wave's shape versus its own.

A

Tight shadow (80% main / 20% own)

Layers look like dark parallax copies of the main wave, slightly horizontally shifted. Every crest above has a crest below. Movement is strongly synchronized.

Feels: silhouette layers, unified motion. Think Studio Ghibli rim-lit ocean at night.

B

Loose shadow (60% main / 40% own) ← recommended

Each layer has its own harmonic variations layered on top of the main wave's primary rhythm. Crests and valleys still line up roughly, but each layer breathes differently — some crests slightly ahead or behind, small ripples that aren't in the main wave.

Feels: real ocean parallax. Layers are related but each has personality.

C

Drift (30% main / 70% own)

Layers have their own rhythm, only loosely biased toward the main wave's shape. You'll often see a main-wave crest with a foreground valley underneath. Motion feels more chaotic and busy.

Feels: choppy, less unified. Probably not what you described.

Technical note: "X% main" means the main wave's surface height (before the Stokes nonlinearity) is blended into each layer's own sinusoidal computation at that weight. Layer then adds its own downward baseline offset and amplitude. Color shade darkens per-layer; a phase offset in tick-space keeps scroll speeds subtly different.
