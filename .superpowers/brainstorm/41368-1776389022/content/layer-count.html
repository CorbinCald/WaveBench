How many wave layers feel right?

Each extra layer is drawn in front of (overlapping) the main wave, positioned lower, with a darker shade of the theme color. More layers = more depth, but more visual noise and CPU cost.

A

1 extra layer (2 total)

Main wave + one foreground shadow wave. Subtle depth cue. Minimal visual noise.

B

2 extra layers (3 total) ← recommended

Main + mid-shadow + near-shadow. Clear parallax. Each shade darker than the last. Good balance of richness and clarity.

C

3 extra layers (4 total)

Lush multi-layer parallax. Each subsequent layer is progressively darker and lower. Richest depth, most visually active.
